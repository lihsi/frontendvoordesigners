# frontend voor designers - opdracht 2

De opdracht is iets maken met 2 UI handlers. Ik heb gekozen voor click en keyboard. 

Het eerste idee was een game waarbij je een plant kon laten groeien door op de knoppen te klikken. Hierbij kon je de achtergrond veranderen met het toetsenbord om de game te customizen. 
Dit is later veranderd naar een game waarbij je de knoppen ook kunt bedienen via het toetsenbord 1/2/3 of Enter voor restart. Zodat er 1 duidelijke hoofdfunctie is. 

Principe 06. 
Er is maar 1 actie op het scherm namelijk op de knoppen drukken om de plant te laten groeien. 

Principe 08. 
De gebruiker weet dat de volgende stap is om op een andere knop te drukken doordat de knop disabled is. Met elke klik groeit de plant en progressbalk

Principe 14. 
Alleen de nodige informatie staat op het scherm.
